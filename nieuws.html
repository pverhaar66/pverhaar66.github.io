Login VSI verbetert Login PI

Login VSI komt met een nieuwe versie van zijn oplossing Login PI. Deze bevat nieuwe netwerkfuncties, beheermogelijkheden, user interface-componenten en waarschuwingsfuncties die voor verbeteringen zorgen van de centrale prestatiebewaking voor virtual desktop infrastructure (vdi) en server based computing (SBC).
